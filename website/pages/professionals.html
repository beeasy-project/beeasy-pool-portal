EasyPool Proxy

Разработанный нами EasyPool Proxy позволет оптимизировать работу множества ригов, находящихся в одной локальной сети за счет быстрого распределения заданий по протоколу Stratum, вести мониторинг состояния ригов и упарвлять ими через панель администратора или бот Telegram

Опции EasyPool Proxy

Высокая скорость распределения заданий

Оптимизация времени работы майнеров, отсутсвие простоев

Мониторинг состояния ригов (хэшрейт/температура/скорость вентиляторов)

Оповещения о внештатных ситуациях с помощью Telegram

Управление оборудованием с помощью Telegram

EasyPool Proxy. Релиз в Сентябре 2017. Поставляется в форме ПО с открытым кодом и доступен на GitHub для самостоятельной установки и в виде предустановленного образа ОС Ubuntu или контейнера виртуальной машины Oracle VirtualBox
